//8 data type in javascript //Number (int) //String (String) //BigInt (BigInt ဆိုတာ JavaScript ထဲမှာ အလွန်ကြီးတဲ့ integer (ကိန်းပြည့်) တန်ဖိုးတွေကို ကိုင်တွယ်နိုင်ဖို့ ECMAScript 2020 (ES11) မှာ မိတ်ဆက်ထားတဲ့ data type တစ်ခုပါ။ BigInt ကဘာလုပ်ပေးသလဲ? JavaScript ရဲ့ Number type က 2^53 - 1 (ဒါက 9007199254740991) ထိပဲ စွမ်းဆောင်နိုင်ပါတယ်။ အဲ့ထက်ကြီးတဲ့ ကိန်းတွေကို loss မရှိဘဲသိမ်းဖို့ BigInt ကအသုံးဝင်ပါတယ်။ ၁။ n တစ်လုံးနဲ့အဆုံးမှာရေးခြင်း: const bigNumber = 1234567890123456789012345678901234567890n; ၂။ BigInt() constructor ကိုသုံးခြင်း: const bigNumber = BigInt("1234567890123456789012345678901234567890"); --- BigInt နဲ့ပတ်သက်တဲ့ သတိပြုရမယ့်အချက်များ: BigInt နဲ့ Number ကို direct addition, comparison လုပ်မရဘူး (TypeError ဖြစ်တတ်တယ်): const a = 10n; const b = 10; console.log(a + BigInt(b)); // OK console.log(Number(a) + b); // Ok --- အသုံးအများဆုံး Use Case ငွေကြေးထဲက အတိအကျသိမ်းဖို့ Blockchain, cryptography ကဲ့သို့သော နယ်ပယ်တွေမှာ အကြီးမားဆုံးသော integer တွေနဲ့ အလုပ်လုပ်ရာမှာ ---------------------------------------------------------------------------------------------------------------------------- //Boolean Boolean ဆိုတာ JavaScript ထဲမှာ true နဲ့ false ဆိုတဲ့ နှစ်ခုတည်းသော တန်ဖိုး သာရှိတဲ့ primitive data type တစ်ခုပါ။ ဒါကို decision making (if, while, for, etc.) တွေမှာ အဓိကအသုံးပြုပါတယ်။ -- Boolean ကိုသုံးတဲ့ နည်းလမ်း ၃ မျိုး ၁။ တန်းတန်းရေးသားခြင်း let isAvailable = true;
